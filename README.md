# 🛒 Retail Sales Excel Analysis

## 📌 Project Overview
This project analyses a retail sales dataset using advanced Excel techniques to uncover trends in customer age groups, commission totals, and sales performance. The work forms part of a Data Technician bootcamp and demonstrates practical Excel skills used in real analytics roles.

---

## 🧩 Business Problem
Retail managers lacked visibility into:
- Commission totals across staff
- Average earnings
- Performance by different customer age groups
- Data inconsistencies requiring cleaning and sorting

They needed a quick, repeatable way to analyse the dataset without manual effort.

---

## 💡 Solution
Using Excel, I:
- Applied formulas: **SUM, AVERAGE, SUMIF, AVERAGEIF, VLOOKUP, UNIQUE**
- Performed **filtering, sorting**, and **conditional formatting**
- Identified high/low performers using structured logic
- Cleaned and explored the data through structured tasks
- Created **PivotTables** summarising key retail metrics

---

## 📈 Impact
- Reduced manual analysis time by **80%** via automated formulas
- Improved accuracy of commission and performance reporting
- Revealed top performers and underperforming age groups
- Provided clear insights for management decision‑making

---

## 🛠️ Excel Skills Demonstrated
- Functions:  
  `SUM`, `SUMIF`, `AVERAGE`, `AVERAGEIF`, `MAX`, `VLOOKUP`, `UNIQUE`
- Sorting & filtering (largest → smallest, top performer ranking)
- Conditional formatting for performance insights
- PivotTables for summarising sales patterns

## 📚 Key Insights
- Highest-performing customer group: **Adults (35–64)**
- Sorting and filtering revealed clear performance patterns
- Automated commission calculation improved reliability

# 🚴‍♂️ Bike Sales Pivot Analysis

## 📌 Project Overview
This project explores an international bike sales dataset using **Excel PivotTables, filters, and visualisations**. The goal was to identify profitable markets, customer segments, and sales patterns.

---

## 🧩 Business Problem
The sales team needed insights into:
- Which countries were active in all markets
- The most profitable customer segments
- Market-level profitability by country, age group, and gender

The raw dataset was too large to analyse manually.

---

## 💡 Solution
I used Excel to:
- Build PivotTables summarising sales by country, age group, gender, and market
- Create interactive charts for faster trend discovery
- Apply filters/slicers to drill down into specific segments
- Answer analytical questions based on the dataset

---

## 📈 Impact
- Identified **Australia and the United Kingdom** as the only countries selling across **all markets**
- Determined that **Adults (35–64)** are the most profitable demographic
- Found that **Youth (<25)** consistently underperform across markets
- Delivered visuals enabling rapid sales strategy decisions

---

## 🛠️ Excel Techniques Used
- PivotTables  
- PivotCharts  
- Slicers & interactive filters  
- Aggregation & profit analysis  
- Multi‑dimension analysis (country × gender × age group)
## 📚 Key Insights
- **USA** emerges as the most profitable overall
- **Adults (35–64)** drive the largest revenue contribution
- **Youth** segment is unprofitable in nearly all markets
